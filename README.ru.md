## 📖 Описание

BandcampDownloader — это приложение Windows, которое помогает загружать альбомы с [bandcamp.com](https://bandcamp.com), извлекая MP3-файлы с битрейтом 128 кбит/с, транслируемые на сайте. Оно призвано облегчить жизнь тем, кто предпочитает слушать музыку на своем любимом плеере, а не в веб-браузере, но предлагает только то, что уже доступно бесплатно на Bandcamp.

## ✨ Функции

### 🎯 Основные функции

| Функция | Описание |
|---------|-------------|
| **📥 Загрузка MP3** | Со страниц альбомов, треков или исполнителей |
| **🏷️ ID3-теги** | Альбом, исполнитель, исполнитель альбома, название, номер трека, год, тексты песен |
| **🖼️ Обложка альбома** | Сохранение в теги и/или папку |
| **📋 Плейлисты** | Форматы M3U, PLS, WPL, ZPL |
| **🎭 Поддержка Various Artists** | Автоматическое разделение формата "Исполнитель - Название" |
| **🔄 Устойчивая загрузка** | Продолжает загрузку при ошибках |
| **⚙️ Персонализация** | Расширенные настройки, включая раздел BDF |

### 🔧 Настройки BDF (эксклюзивно для BandcampDownloaderFixed)

> Раздел **BDF** предоставляет параметры надежности и удобства для улучшенного опыта загрузки

| Функция | Описание |
|---------|-------------|
| **✅ Продолжить при ошибке** | Продолжает загрузку оставшихся альбомов/треков при возникновении ошибок |
| **⏳ Ожидание готовности файла** | Обрабатывает условия гонки, ожидая полной записи файлов |
| **🎨 Разделение заголовков V/A** | Правильно обрабатывает релизы Various Artists |
| **⏱️ Игнорировать длинные треки** | Пропускает треки, превышающие указанную длительность |
| **👤 Назначить исполнителя альбома** | Устанавливает исполнителя альбома для всех треков (для сборников) |
| **📊 Показать количество треков** | Отображает прогресс загрузки в панели действий |
| **🎚️ Принудительный битрейт** | Экспериментальное преобразование битрейта |
| **📏 Показать исходный размер** | Отображает исходный и преобразованный размер файла |
| **🔔 Уведомления Toast** | Уведомления Windows для событий загрузки |
| **🐛 Подробное диалоговое окно ошибок** | Показывает подробную информацию об ошибках |

## ⚠️ Примечание о пиратстве

> Вы можете делать с этим приложением что хотите, но помните, что если вы хотите поддержать своих любимых исполнителей, **покупайте их альбомы**! 💜

### Позиция Bandcamp

Из [официального заявления Bandcamp](https://get.bandcamp.help/hc/en-us/articles/23020694039575-I-heard-you-can-steal-music-on-Bandcamp-What-are-you-doing-about-this):

> **Один из моих фанатов показал мне совершенно простой способ, как кто-то может УКРАСТЬ мою музыку с Bandcamp, используя RealPlayer 14.1 beta 3, RipTheWeb.com или зайдя во временные файлы Интернета и переименовав бла-бла-бла. Что вы собираетесь делать с этой серьезной проблемой?**
>
> Ничего. Поскольку потоки на Bandcamp имеют полную длину, действительно возможно, что кто-то может использовать один из вышеперечисленных методов для доступа к базовому MP3-128. И конечно, мы могли бы установить технические препятствия, но если они наткнутся на эти препятствия, они не ударятся себя лбом и откроют кошелек. Вместо этого они перейдут на другой сайт, где таких ограничений нет, и вы упустите шанс сделать свой сайт главным направлением для этих, казалось бы, дешевых, но энтузиастичных, распространяющих информацию и потенциально тратящих деньги фанатов. Другими словами, немногие люди, использующие вышеперечисленные методы, следует рассматривать как возможность, а не потерянную продажу. Если вы все еще сомневаетесь, [это может помочь](https://newmusicstrategies.com/but-if-they-steal-it/).
